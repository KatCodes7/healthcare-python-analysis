# Healthcare Data Analysis with Python

This project analyzes healthcare claims data to identify cost drivers, demographic trends, and the impact of chronic conditions on healthcare spending.

## Tools Used
- Python (pandas, matplotlib)
- Jupyter Notebook

## Project Overview

This analysis explores how factors such as age and chronic conditions influence healthcare reimbursement patterns.

Key steps included:
- Data cleaning and preprocessing
- Feature engineering (age calculation from DOB)
- Outlier handling
- Aggregation and grouping
- Data visualization
- Insight generation

## Key Insights

### Age-Based Trends
- Reimbursement was lowest for beneficiaries under 50  
- Highest average costs were observed in the 50–75 age range  
- Costs did not increase consistently across all age groups  

### Chronic Condition Impact (Diabetes)
- Beneficiaries with diabetes had significantly higher average reimbursement  
- Diabetes is a major driver of healthcare costs in this dataset
